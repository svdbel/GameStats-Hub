О проекте GameStats Hub

Игровой информационный портал для статистики Dota 2, рейтингов игроков и аналитики.

Технологический стек123:

  • Веб-приложение: Python Flask, HTML/CSS/JavaScript, Bootstrap
  • API интеграция: OpenDota API
  • Контейнеризация: Docker, Docker Compose
  • CI/CD: GitHub Actions
  • Инфраструктура как код: Terraform, Ansible
  • Мониторинг: Prometheus, Grafana, AlertManager, cAdvisor
  • Логирование: ELK Stack (Elasticsearch, Kibana, Logstash, Filebeat)
  • Облачная платформа: Google Cloud Platform (GCP)
  • CDN и безопасность: Cloudflare
  • Веб-сервер: Nginx Reverse Proxy

Доступ к сервисам:

  • Основное приложение: https://gamestats.svdbel.org
  • Мониторинг (Grafana): https://grafana.svdbel.org
  • Логи (Kibana): https://kibana.svdbel.org
  • Метрики (Prometheus): https://prometheus.svdbel.org

Исходный код: GitHub Repository
